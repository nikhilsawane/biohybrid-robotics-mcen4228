# MCEN 4228/5228 – Biohybrid Robotics (Spring 2026)

This repository documents my coursework, labs, notes, and projects for  
**MCEN 4228/5228: Biohybrid Robotics** at the University of Colorado Boulder.

The course explores the integration of living systems with engineered robotic platforms, including:
- Microorganism-based robots
- Tissue-engineered muscle systems
- Cyborg insects and animals
- Biohybrid sensing and actuation

## Repository Structure
- `lectures/` – Lecture notes and summaries
- `guest-seminars/` – Paper notes and discussion questions
- `labs/` – Structured labs and group projects
- `final-project/` – Proposal, design, results, and report
- `code/` – Arduino, Python, and MATLAB code
- `reflections/` – Weekly and final reflections
